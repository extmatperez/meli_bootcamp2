¿A qué se denomina JOIN en una base de datos?
En la combinacion de dos tablas de datos mediante 
la relacion de no o varias condiciones en comun, 
para una misma consulta.

Nombre y explique 2 tipos de JOIN.
Inner Join: Corresponde a la seleccion de datos
con elementos en comun por ambos.

RIGTH JOIN: Corresponde a la seleccion de datos
que posee una tabla B mas los datos en comun para
las tablas A y B

¿Para qué se utiliza el GROUP BY?
Para agrupar los reultados segun un registro que se
requiera, para obtener datos resumidos que esten
agrupados por las columnas que se necesiten.

¿Para qué se utiliza el HAVING?
Para poder filtrar luego de realizar un group by, 
esto debido a que el where solo se puede ocupar 
antes del group.
